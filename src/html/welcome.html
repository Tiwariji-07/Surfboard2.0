Welcome to WaveMaker Copilot

Your AI-powered assistant for WaveMaker development

Features

  • Intelligent context-aware assistance for WaveMaker development
  • Real-time suggestions and best practices
  • Code snippets and component recommendations
  • Natural language interaction with AI
  • Seamless integration with WaveMaker Studio

Getting Started

  1. Set up your API key

    You'll need an OpenAI API key to use WaveMaker Copilot. Get your key from:

    OpenAI API Dashboard
  2. Configure the extension

    Click the extension icon and enter your API key when prompted.

  3. Start using Copilot

    Open WaveMaker Studio and press Ctrl + \ to toggle the Copilot sidebar.

Keyboard Shortcuts

  • Ctrl + \ - Toggle Copilot sidebar
  • Ctrl + Enter - Send message in chat
  • Esc - Close current suggestion
